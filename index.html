Tentang Saya

Saya seorang Direktur Kreatif dan Desainer UI/UX dari Pringsewu, Lampung , yang bekerja di bidang pengembangan web dan media cetak. Saya senang mengubah masalah rumit menjadi desain yang sederhana, indah, dan intuitif.

Tugas saya adalah membangun situs web Anda agar fungsional dan ramah pengguna namun sekaligus menarik. Selain itu, saya menambahkan sentuhan pribadi pada produk Anda dan memastikannya menarik serta mudah digunakan. Tujuan saya adalah untuk menyampaikan pesan dan identitas Anda dengan cara paling kreatif. Saya telah membuat desain web untuk banyak perusahaan merek terkenal.
